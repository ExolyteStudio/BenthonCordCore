# Image Placeholders

This folder contains placeholder locations for your website images.

## Required Images:

### Root Directory:
- **logo.png** - Your BenthonCord logo (recommended: 200x50px, transparent background)
- **icon.png** - Website favicon (recommended: 64x64px or 128x128px)

### Tutorial Images (images/ folder):
Replace these with actual screenshots from Discord Developer Portal:

1. **step1.png** - Discord Developer Portal homepage
2. **step2.png** - "New Application" button location
3. **step3.png** - Application name input dialog
4. **step4.png** - Create button
5. **step5.png** - Application ID location on dashboard
6. **step6.png** - Copy button for Application ID
7. **step7.png** - Rich Presence assets upload (optional step)
8. **step8.png** - BenthonCord configuration screen

## Image Guidelines:

- Use PNG format for transparency (logo, icon)
- Use JPG or PNG for tutorial screenshots
- Keep file sizes reasonable (< 500KB each)
- Ensure images are clear and readable
- Use consistent dimensions for tutorial images
- Consider adding borders or highlights to important UI elements in screenshots

## How to Add Images:

1. Create or capture your images
2. Name them exactly as listed above
3. Place them in the correct directories
4. Test the website to ensure they display correctly

Currently, the website uses placeholder styling for tutorial images. Once you add real images, they will automatically appear in the guide.
