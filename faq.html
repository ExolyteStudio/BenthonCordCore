Frequently Asked Questions

Find answers to the most common questions about BenthonCord

What is BenthonCord?

+

BenthonCord is an advanced Discord Rich Presence (RPC) management client that allows you to fully customize and control your Discord status. With BenthonCord, you can create custom presence displays, manage multiple profiles, and showcase what you're doing in real-time on Discord.

Is BenthonCord free to use?

+

Yes! BenthonCord is completely free to download and use. All features are available at no cost, and there are no hidden fees or premium tiers.

Is BenthonCord safe to use?

+

Absolutely! BenthonCord is built with security and privacy as top priorities. The application only uses official Discord APIs and never stores or transmits your personal data. Your Discord token is handled securely and never leaves your device.

What operating systems are supported?

+

BenthonCord supports Windows (x64 and x86), macOS (Intel and Apple Silicon), and Linux (Debian/Ubuntu and AppImage). Check our download page for specific version requirements.

How do I get my Discord Application ID?

+

To set up custom RPC, you need a Discord Application ID. We've created a detailed step-by-step guide for this process. Click here to view the complete tutorial on how to obtain your Application ID from the Discord Developer Portal.

Can I use BenthonCord with multiple Discord accounts?

+

Yes! BenthonCord version 3.2.1 and later includes full support for managing RPC across multiple Discord accounts simultaneously. You can create separate profiles for each account.

Why isn't my Rich Presence showing up?

+

There are several common reasons:

  • Make sure Discord is running on your computer
  • Check that your Application ID is correct
  • Ensure "Display current activity as a status message" is enabled in Discord settings
  • Verify that BenthonCord has the necessary permissions
  • Try restarting both BenthonCord and Discord

How do I update BenthonCord?

+

BenthonCord includes an automatic update feature that will notify you when a new version is available. You can also manually check for updates in the application settings or download the latest version from our download page.

Can I save multiple RPC configurations?

+

Yes! BenthonCord includes a Profile Presets feature that allows you to save unlimited RPC configurations and switch between them with a single click. This is perfect for different activities, games, or moods.

Does BenthonCord work with Discord Nitro?

+

BenthonCord works with all Discord accounts, whether you have Nitro or not. Discord Nitro is not required to use BenthonCord or Rich Presence features.

Will using BenthonCord get me banned from Discord?

+

No. BenthonCord only uses official Discord APIs and follows all Discord Terms of Service. Rich Presence is an officially supported Discord feature, and using BenthonCord to manage it is completely safe and legitimate.

How do I uninstall BenthonCord?

+

On Windows, use the standard "Add or Remove Programs" feature in Settings. On macOS, drag the application to the Trash. On Linux, use your package manager's uninstall command or delete the AppImage file.

I found a bug. How do I report it?

+

We appreciate bug reports! Please visit our projects page to find our support channels and submit a detailed bug report including your OS version, BenthonCord version, and steps to reproduce the issue.

Can I contribute to BenthonCord development?

+

We welcome contributions! Visit our projects page to learn more about contributing to BenthonCord, whether through code, translations, documentation, or feature suggestions.

Still need help?

If you couldn't find the answer you're looking for, visit our projects page for additional support options and community forums.
